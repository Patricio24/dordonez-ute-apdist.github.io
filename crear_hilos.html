APLICACIONES DISTRIBUIDAS

Creación de hilos

En Java los hilos están encapsulados en la clase Thread. Para crear un hilo tenemos dos posibilidades:

Extender Thread

  • Crea una clase T que extiende de Thread
  • Sobreescribir el metodo run()
    • Incluir ahi el codigo que debe ejecutarse en el nuevo hilo

  • Crear una instancia t de T
  • Ejecutar t.start()

    • Se crea un nuevo hilo (path de ejecucion)
    • Dentro de dicho hilo se ejecuta run()
    • Al finalizar run(), finaliza hilo

Crear un hilo heredando o Extender Thread tiene el problema de que al no haber herencia múltiple en Java, si heredamos de Thread no podremos heredar de ninguna otra clase, y por lo tanto un hilo no podría heredar de ninguna otra clase.

Implementar Runnable

  • Crea una clase R que implemente Runnable
  • Sobreescribe el metodo run()
    • Incluir ahi el codigo que debe ejecutarse en el nuevo hilo

  • Crear una instancia r de R
  • Pasar r al constructor de una instancia t de Thread: Thread t=new Thread(r);
  • Ejecutar t.start()

    • Se crea un nuevo hilo (path de ejecucion)
    • Dentro de dicho hilo se ejecuta run()
    • Al finalizar run(), finaliza hilo

!!!En ambos casos debemos definir un método run() que será el que contenga el código del hilo!!!
