INTRODUCCION

El Lenguaje de Marcas de Hipertexto o HyperText Markup Language (HTML) es el lenguaje de marcado para la elaboración de páginas web. Su más reciente versión es HTML5, un lenguaje más avanzado concentrado en brindar una gran flexibilidad para estructurar y presentar páginas Web.

La Hojas de Estilo en Casacada o Cascading Style Sheets (CSS) es un lenguaje usado para definir la presentación de un documento escrito en HTML. El objetivo principal es separar la estructura de un documento de su presentación. Su última versión es el CSS3. El lenguaje CSS3 se basa en una serie de reglas (que forman la sintaxis de las hojas de estilo) que rigen el estilo de los elementos en los documentos estructurados. Con estas reglas se puede seleccionar el color de fondo o del texto, indicar cuál va hacer el tipo de letra (negrita o cursiva) que se va a utilizar o definir el tamaño de las letras o de los elementos (por ejemplo, el alto y el ancho de una tabla).

Degradados en CSS3

Un color que se va degradando progresivamente hasta convertirse en otro es un efecto muy usado para fondos de elementos (o incluso como fondo web) ya que permite conseguir un efecto atractivo. Este efecto antiguamente no era fácil de conseguir y se solía recurrir al uso de imágenes de fondo. Actualmente disponemos de mecanismos que facilitan incluir degradados en páginas web.
